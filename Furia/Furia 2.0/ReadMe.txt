Furia pesquisa
---------------------------
Funcionalidades:

-Mensagem Automatica de Boas-Vindas
-Suporte para o Botão Enter
-Menu de Links para outras paginas da Furia, inclusive o Chatbot
-Link Cliclaveis para a Loja oficial e redes sociais da Furia
-Facilidade de modelar o chat para conter novas perguntas e possibilidade de adição de um banco de dados para coleta mais eficiente
-Tema da pagina com as Cores da Furia
-A Cor das letras tanto do Bot como do Usuario são bem visiveis
-Botão para ir para o site oficial da Furia
----------------------------
Como usar:

1. Salvar na mesma pasta os arquivos:
	-index2.html
	-index.css
	-data.js
	-Furia_logo.png
    -index - Atalho.lnk
2. Abrir o index2.html utilizando um navegador
